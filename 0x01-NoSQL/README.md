### Learning Objectives:

 - What is ACID
 - What is a document storage
 - What are NoSQL types
 - What are benefits of a NoSQL database
 - How to query information from a NoSQL database
 - How to insert/update/delete information from a NoSQL database
 - How to use MongoDB
